# My Personal Portfolio Website 🚀

Welcome to the repository for my personal portfolio website! This project is a clean, beautiful, and fully responsive website built to showcase my skills, experiences, projects, and achievements.

---

## 📝 About

This portfolio acts as a central hub for my professional identity online. It dynamically aggregates my open-source footprints and highlights my career journey. Key features include:

- **Dynamic GitHub Integration:** Automatically fetches pinned repositories and open-source contributions using the GitHub GraphQL API.
- **Interactive UI:** Smooth scrolling, clean animations, and responsive layouts designed to look great on mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

The architecture of this application relies on modern web technologies and lightweight animation libraries:

- **Core Framework:** React.js, HTML5, CSS3, Sass
- **Data Fetching:** GraphQL (via GitHub API)
- **Animations:** React-Reveal, Lottie Files, Framer Motion
- **Icons & UI Graphics:** FontAwesome, Iconify, custom SVGs

---

## ⚙️ Installation & Setup

Follow these steps to get the project up and running on your local machine for development:

### Prerequisites

Ensure you have **Node.js** (v14 or higher recommended) and **npm** installed on your system.

### 1. Clone the Repository

```bash
git clone [https://github.com/](https://github.com/)<your-github-username>/<your-repo-name>.git
cd <your-repo-name>
```

### 2. Install Dependencies

Install all the required npm packages specified in the project configuration:

```bash
npm install
```

### 3. Run the Development Server

Start the local server to preview the website in your browser:

```bash
npm start
```

The application will open automatically at http://localhost:3000.
